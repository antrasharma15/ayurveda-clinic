### Option 1: Automatic Installation (Recommended)
```bash
cd /opt/lampp/htdocs/ayurveda-clinic-main\(2\)/ayurveda-clinic-main
./install.sh
```

### Option 2: Manual Installation
```bash
# Step 1: Update Database
/opt/lampp/bin/mysql -u root ayurveda_clinic < update_db.sql

# Step 2: Set Permissions
chmod -R 777 images/uploads

# Step 3: Done! Go to admin panel
```

==============================================
🚀 HOW TO USE
==============================================

1. Login to Admin Panel
2. Go to "Manage Menus"
3. Fill the form:
   - Menu Name: e.g., "Our Services"
   - URL/File Name: e.g., "our-services.php"
   - Check ✅ "Create Custom Page"
   - Fill SEO fields (optional)
4. Click "Add Menu & Create Page"
5. Click Edit (✏️) to add content with rich text editor
6. Visit your new page!

==============================================
🐛 TROUBLESHOOTING
==============================================

Problem: Images not uploading
Solution: chmod -R 777 images/uploads

Problem: Database error
Solution: Run update_db.sql again

Problem: PHP file not created
Solution: Check "Custom Page" checkbox

Problem: Editor not loading
Solution: Check internet connection (TinyMCE CDN)
